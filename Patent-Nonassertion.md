# Patent Non-Assertion Pledge

## Constitutional Commons Patent Protection

Any contributor to this repository grants irrevocable, worldwide, royalty-free 
patent license to any entity using this software for constitutional governance 
purposes, provided such use complies with the GNU Affero GPL v3 license.

## No Patent Aggression
Contributors covenant not to assert any patent claims against users, contributors, 
or distributors of this software when used in compliance with the Constitutional 
Commons principles.

## Scope
This pledge covers all patent claims licensable by the contributor that would 
be infringed by the software in its unmodified form.
